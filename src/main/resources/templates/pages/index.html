Welcome to Drink Manager

Drink Manager your go-to solution for efficient beverage collection management, developed as part of the ADT diploma project.

Designed specifically as an admin dashboard, Drink Manager simplifies the task of organizing your drink inventory. With our user-friendly interface, you can easily catalog various beverages, track quantities, and manage expiration dates all in one place.

You can find the source code on our GitHub repository: AutomatischerDosenTransporter
